Exploring Elixir [Keyword lists](https://elixir-lang.org/getting-started/keywords-and-maps.html) as options to functions. This was inspired by lecture 15 in Udemy course [Elixir for Beginners](https://www.udemy.com/course/elixir-for-beginners/) by [Ben Emson](https://www.udemy.com/user/benemson/)

```shell
PS keyword_lists_as_function_options> elixir .\main.exs 
deep fried hamburger
deep fried taters
deep fried pizza

Option sauce: garlic
Option topping: basil
Option joiner: with
Today we are serving a nice garlic sauce, some pizza with cheese and a sprinkle of basil

Unknown option im_hungry_now: mmm foood
Today we are serving ice cream and brownes and popcorn

Option sauce: ketchup
Today we are serving a nice ketchup sauce, some pasta and meatballs
```
